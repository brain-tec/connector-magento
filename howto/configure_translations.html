How to configure translations¶

If you have different translations on Magento, say one store view displayed in english and a second in french, you have to configure the languages to use in Odoo.

  1. Set the default language in the Magento Backend (Connectors > Magento > Backend) in the field Default Language.
  2. Then, for each storeview (Connectors > Magento > Storeviews), change the language of the storeview.
